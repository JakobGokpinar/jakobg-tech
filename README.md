# jakobg.tech

My personal portfolio website built with Next.js 15, React 19, Tailwind CSS v4, and Motion-Primitives.


## Stack

- **Framework:** Next.js 15 + React 19
- **Styling:** Tailwind CSS v4
- **Animations:** [Motion-Primitives](https://motion-primitives.com)
- **Blog:** MDX
- **Hosting:** Vercel
- **Template:** Based on [Nim](https://github.com/ibelick/nim)
